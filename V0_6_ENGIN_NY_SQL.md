# Námsmiðjan v0.6

v0.6 þarf ekki nýtt SQL schema ef v0.5 er þegar keyrt.

Notaðu áfram sömu töflur:
- namsmidjan_modules
- namsmidjan_tasks
- namsmidjan_progress

V0.6 bætir aðallega við UI/kennaramælaborði og nýjum stöðum í framvindu:
- help
- challenge

Þessar stöður vistast í sama `status` dálki í `namsmidjan_progress`.
