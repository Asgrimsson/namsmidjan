Hnappurinn **Laga Lestarþjálfun flís** setur:
- Heiti: Lestarþjálfun
- Fag: Íslenska
- Bekkir: 1.–6.
- Lýsing: Lestarverkefni, orðaforði, lesskilningur og skráning á framförum.

## Uppfærsla

```bash
cd C:\Namsmidjan
git add .
git commit -m "Laga fog og Lestarthjalfun flis v1.2.1"
git push
```

Eftir deploy:
1. Opnaðu Námsmiðjuna.
2. Farðu í Vefjasafn.
3. Smelltu á Laga Lestarþjálfun flís.
4. Farðu í Gagnagrunnur.
5. Smelltu á Senda local gögn í Supabase.
