# Námsmiðjan v1.4 — Bekkir og nemendaskrá

## Uppfæra vefinn

```bash
cd C:\Namsmidjan
git add .
git commit -m "Baeti vid bekkjum og nemendaskra v1.4"
git push
```

## Keyra SQL

Í Supabase SQL Editor skaltu keyra:

```text
supabase_v1_4_classes_students.sql
```

Þetta býr til:
- `namsmidjan_classes`
- `namsmidjan_students`

## Nota kerfið

1. Opnaðu Námsmiðjuna.
2. Farðu í **Bekkir**.
3. Smelltu á **Nýr bekkur**.
4. Vistaðu bekk.
5. Límdu inn nemendalista.
6. Smelltu á **Búa til nemendur**.
7. Smelltu á **Vista bekki í Supabase**.
