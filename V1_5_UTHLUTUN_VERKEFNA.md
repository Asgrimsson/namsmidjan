# Námsmiðjan v1.5 — Úthlutun verkefna á bekki og hópa

## 1. Uppfæra vefinn

```bash
cd C:\Namsmidjan
git add .
git commit -m "Baeti vid uthlutun verkefna v1.5"
git push
```

## 2. Keyra SQL í Supabase

Opnaðu Supabase → SQL Editor.

Keyrðu skrána:

```text
supabase_v1_5_assignments.sql
```

Þetta býr til töfluna:

```text
namsmidjan_assignments
```

## 3. Nota úthlutun

1. Búðu til verkefni í AI verkefnasmið eða verkefnasmið.
2. Vistaðu verkefnapakkann.
3. Farðu í **Úthlutun**.
4. Veldu verkefni/pakka.
5. Veldu bekk/hóp.
6. Settu skiladag og fyrirmæli.
7. Smelltu á **Úthluta verkefni**.
8. Smelltu á **Vista úthlutanir**.

## 4. Hvernig nemendur nota þetta núna?

Í v1.5 er úthlutunin komin í gagnagrunninn. Nemendur geta enn notað verkefnakóða.

Næsta skref, v1.6, er að láta Nemendagáttina sýna sjálfkrafa verkefni sem eru virk fyrir bekkinn þeirra.
