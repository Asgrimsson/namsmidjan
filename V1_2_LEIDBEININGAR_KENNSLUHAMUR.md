# Námsmiðjan v1.2 — Leiðbeiningar og kennsluhamur

## Uppfæra á Netlify

1. Afþjappaðu ZIP.
2. Farðu inn í `namsmidjan_v0_1`.
3. Afritaðu allt yfir `C:\Namsmidjan`.
4. Keyrðu:

```bash
cd C:\Namsmidjan
git add .
git commit -m "Baeti vid leidbeiningum og kennsluham v1.2"
git push
```

Netlify deployar sjálfkrafa.
