# Námsmiðjan v1.1 — Vefjasafn Vallaskóla

## Uppfærsla
1. Afþjappaðu ZIP.
2. Afritaðu allt úr `namsmidjan_v0_1` yfir `C:\Namsmidjan`.
3. Keyrðu:
```bash
cd C:\Namsmidjan
git add .
git commit -m "Baeti vid Vefjasafni Vallaskola v1.1"
git push
```

## Virkja inni í vefnum
1. Opnaðu https://namsmidjan.netlify.app
2. Smelltu á **Vefjasafn**
3. Smelltu á **Setja inn / uppfæra vefjasafn**
4. Farðu í **Gagnagrunnur**
5. Skráðu þig inn sem kennari
6. Smelltu á **Senda local gögn í Supabase**

Þá fara flísar í `namsmidjan_modules` og verkefnaspjöld í `namsmidjan_tasks`.
